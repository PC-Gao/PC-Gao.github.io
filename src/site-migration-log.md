# 网站迁移与更新记录

本文记录个人主页从旧站点结构迁移到当前 `mdBook` 结构的过程，以及目前已经完成的主要调整，便于后续继续维护。

## 迁移背景

原仓库基于旧的 GitHub Pages / Jekyll 风格模板，内容里混有不少并不属于个人主页的模板残留。当前版本已经将站点重构为更适合长期写作和文档整理的 `mdBook` 结构，目标是：

- 用统一的文档站方式管理项目、博客和闲谈
- 保留清晰的左侧章节导航和右侧页内目录
- 让个人主页更适合持续写文章，而不是一次性作品集

## 当前结构

当前内容主要位于 `src/` 下：

- `about.md`：关于页，同时作为站点主入口
- `archive.md`：内容归档
- `projects/`：项目记录
- `blog/`：博客文章
- `notes/`：实验记录、闲谈和杂记
- `SUMMARY.md`：左侧导航目录定义

主题相关定制位于 `theme/`：

- `custom.css`：站点样式覆盖
- `page-toc.js`：右侧页内目录生成逻辑
- `giscus.js`：评论区注入逻辑

## 已完成的迁移工作

### 1. 站点框架迁移

- 从旧模板切换到 `mdBook`
- 清理不属于个人主页的模板残留内容
- 统一为文档站结构，而不是传统博客首页加碎片化页面

### 2. 导航重组

- 将首页和关于页合并为 `About`
- 左侧导航改为站点章节入口，不再混入文章内部标题
- 项目、博客、闲谈分别独立成一级分区
- 新增本页作为维护记录，并放在导航栏最下处

### 3. 页面布局调整

- 布局整体参考 `theajack.github.io/rust` 的文档站形式
- 左侧保留书籍式目录导航
- 右侧目录根据当前 Markdown 标题自动生成
- 右侧目录固定在页面右侧，滚动时高亮当前标题
- 页面底部翻页箭头调整为更接近目标文档站的形式

### 4. 目录样式调整

左侧目录目前采用接近目标站的参数：

- 侧栏字号：`0.875em`
- 一级条目行高：`1.5em`
- 二级目录缩进：`20px`
- 二级目录行高：`1.9em`

右侧目录目前采用接近目标站的参数：

- 目录宽度：`200px`
- 条目字号：`1.2rem`
- 上下内边距：`5px`
- 左内边距：`10px`

### 5. 评论系统迁移

- 原仓库使用 `giscus`
- 当前站点已经将评论区迁移到 `mdBook`
- 评论区会在页面正文底部自动插入，无需每篇文章单独写脚本
- 当前评论主题会随 `mdBook` 主题切换

## 本地预览与构建

### 构建

```powershell
mdbook build
```

### 本地预览

`mdbook serve` 在当前 Windows 环境下有端口绑定问题时，可以使用：

```powershell
mdbook build
python -m http.server 3000 --directory book
```

然后访问：

```text
http://127.0.0.1:3000/
```
